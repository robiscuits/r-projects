  1. Write code to mimic the built in MA plot from DESeq2, but use ggplot2.
  • The columns of resLFC are:
    • baseMean
    • log2FoldChange
    • lfcSE
    • pvalue
    • padj
  • The column padj has adjusted p-values which takes false discovery rate and differential expression into account
  • Your plot should:
    • have log2FoldChange on the y axis
    • have the base ten log of baseMean on the x axis
    • for points outside the limits, use:
      • pch = 2, the up-pointing triangle, for any points above the maximum y limit
      • pch = 6, the down-pointing triangle, for any pints below the minimum y limitn
    • use a period (dot) as the plotting character for points with y-values within the limits
    • color points:
      • black if the adjusted p-value padj is greater than or equal to 0.1
      • red if the adjusted p-value padj is less than 0.1
    • add a red horizontal line with a y-intercept of 0 (and alpha = 0.5)
    • use the theme_bw() black and white theme
    • create readable labels
    • use good tick labels (without scientifi notation)

col_data = tibble( names = names(rna)[-1] ) %>% 
  separate(names, into = c("ecotype","condition","replicate"), remove = FALSE) %>%

## check that names match and are in the same order

## Create the DESeq2 object
## Note the design formula design = ~ ecotype + condition
## The primary factor of interest should come last
dds = DESeqDataSetFromMatrix(countData = cts,
                             colData = col_data,
                             design = ~ ecotype + condition)

## Prefiltering
## The package authors recommend getting rid of any genes where the sum of total counts is small to avoid spurious results where there is little signal and power
## Here, filter to only keep rows where the total counts are 20 or more
## See how many genes this eliminates from the analysis

## before
nrow(dds)
## [1] 41671
keep = rowSums(counts(dds)) >= 20
dds = dds[keep,]

## after
nrow(dds)
## [1] 31483
dds = DESeq(dds)
res = results(dds)


resLFC = lfcShrink(dds, coef = "condition_SpaceFlight_vs_Ground", type = "apeglm")
## reorder genes from smallest to largest pvalue
resLFC = resLFC[order(resLFC$pvalue),]
